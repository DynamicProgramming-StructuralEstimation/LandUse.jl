LandUse.jl documentation

Intro

This is the documentation for julia computation code used in Structural Change, Land Use and Urban Expansion by Coeurdacier, Oswald and Teignier. The package repository is on github at https://github.com/floswald/LandUse.jl.

There is an associated R package LandUseR whose website, documents for the data measurement parts used in the paper, basically the code for appendix B.6 and B.7.
